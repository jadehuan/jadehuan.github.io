GitHub Pages 搭建个人博客

2026-05-21 · 实验记录

本文记录了使用 GitHub Pages 搭建个人静态博客的过程。 GitHub Pages 是 GitHub 提供的免费静态网页托管服务， 通过简单的配置即可将个人仓库中的 HTML 文件发布为网站。

绝对路径图片示例

以下图片通过绝对路径（完整 URL）引用自外部图床：

src="https://jadehuan.github.io/image/1.png"
图1：使用绝对路径引用的图片（https://jadehuan.github.io/image/1.png）

绝对路径包含了完整的协议（https）、域名（jadehuan.github.io）和资源路径（/image/1.png）， 无论网页部署在哪里，都能正确加载该图片。

相对路径图片示例

以下图片通过相对路径引用自项目内的文件：

src="../image/1.png"
图2：使用相对路径引用的图片（../image/1.png）—— 我的壁纸

相对路径以当前 HTML 文件所在位置为基准，../ 表示上一级目录， 再进入 image/ 文件夹找到 1.png。 相对路径的优势在于：项目整体迁移时，只要保持内部目录结构不变，图片链接就不会失效。

相对路径 vs 绝对路径

2026-05-21 · 学习笔记
对比维度 绝对路径 相对路径
定义 完整的 URL 地址 相对于当前文件的路径
示例 https://example.com/img/1.png ../image/1.png
优点 不受文件位置影响 便于项目整体迁移
缺点 域名变更则全部失效 文件移动后需调整路径
